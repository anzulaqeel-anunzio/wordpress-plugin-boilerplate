# WordPress Plugin Boilerplate

A clean, object-oriented foundation for building WordPress plugins.

## Features

-   **Standard Structure**: Adherence to WordPress coding standards.
-   **Class-Based**: Object-Oriented Programming (OOP) approach.
-   **Activation/Deactivation Hooks**: Ready-to-use hooks for setup and teardown.
-   **Separation of Concerns**: Admin and Public functionality separated.

## Getting Started

1.  **Installation**
    -   Zip the folder.
    -   Upload to WordPress via Plugins > Add New > Upload Plugin.
    -   Activate.

## Project Structure

-   `anunzio-plugin.php`: Main plugin entry file.
-   `includes/`: Core plugin logic and classes.
-   `admin/`: Dashboard-specific functionality.
-   `public/`: Frontend-specific functionality.
